pfSense — Routeur & Pare-feu

Personnel
Objectifs
  • Déployer un routeur/pare-feu pfSense pour sécuriser mon réseau local
  • Mettre en place des VLANs pour segmenter les flux réseau
  • Configurer un VPN OpenVPN pour accès distant sécurisé
  • Gérer les règles de pare-feu et redirections de ports (NAT)
Actions réalisées
  • Installation de pfSense sur une VM
  • Configuration du routage et du DHCP pour le réseau interne
  • Mise en place de VLANs pour isoler serveurs et postes clients
  • Création d'un serveur OpenVPN pour accès distant à mes machines
  • Configuration de règles de pare-feu pour filtrer le trafic entrant/sortant
Résultats
  • Infrastructure réseau sécurisée avec segmentation VLAN fonctionnelle
  • Accès VPN opérationnel depuis l'extérieur
  • Trafic réseau filtré et contrôlé via règles de pare-feu
  • Documentation complète de l'installation et configuration
Technologies
pfSense FreeBSD OpenVPN VLANs NAT/Firewall
